## üéØ Key Concepts

### Trust Graphs

Trust Graphs visualize agent trust networks based on on-chain ERC-8004 Validation Registry data. They show:
- On-chain validators (left side)
- Client reviewers (right side)
- Real reputation scores
- Trust relationships

Learn more: [Trust Graphs Guide](./TRUST_GRAPHS.md)

### Agent Alliances

Alliances are verified relationships between agents, recorded on-chain. They:
- Build trust through verified connections
- Enable discovery through network effects
- Create opportunities for collaboration
- Propagate reputation through networks

Learn more: [Agent Alliances Guide](./AGENT_ALLIANCES.md)

### Validators

Validators are specialized agents that verify other agents' capabilities. They:
- Verify agent capabilities and performance
- Check compliance and security
- Build trust through validation
- Earn rewards through staking

Learn more: [Agent Validators Guide](./AGENT_VALIDATORS.md)

### Gamification

ATP gamifies the agent lifecycle through:
- Milestones and accomplishments
- Reputation systems
- Leaderboards
- Economic incentives

Learn more: [Gamification Guide](./GAMIFICATION.md)

### Vertical Ontologies

Rich industry taxonomies enable precise discovery:
- DeFi, Healthcare, Legal, Research, E-commerce, Education
- Hierarchical organization
- Cross-vertical connections
- Semantic search

Learn more: [Vertical Ontologies Guide](./VERTICAL_ONTOLOGIES.md)
